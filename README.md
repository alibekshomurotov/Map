# 🌍 Interactive Geography Map

2D globus ko'rinishidagi interaktiv geografik xarita. Dunyoning istalgan davlatini qidirish, ko'rish va ular haqida ma'lumot olish imkoniyati.

## ✨ Xususiyatlar

- 🌐 3D globus ko'rinishi
- 🔍 Davlatlarni qidirish (o'zbek va ingliz tillarida)
- 📍 Davlat poytaxtlariga uchish
- 📊 Davlat haqida ma'lumot (aholi, poytaxt, koordinata)
- 🎨 Zamonaviy va interaktiv dizayn
- 📱 Mobil qurilmalarga moslashtirilgan

## 🚀 Texnologiyalar

- React 18
- Vite
- Leaflet (xarita)
- React-Leaflet
